# Release Notes — 0.0.24

**Paquete:** `ng-magary`  
**Tipo:** `MINOR`  
**Estado:** `Draft — not published`  
**Tag objetivo:** `v0.0.24`

---

## Resumen

`0.0.24` es el primer release gobernado bajo la nueva disciplina de estabilización de Magary.

Este release combina:

- mejoras funcionales acotadas en componentes Data,
- trazabilidad formal de cambios,
- y artefactos de gobernanza para hacer releases repetibles y auditables.

---

## Qué cambia

### Data components

Se introduce un slice funcional mínimo y acotado en:

- `DataView`
- `OrderList`
- `PickList`
- `Tree`

En estos componentes se fortalecen estados asíncronos y de error para hacer el consumo más confiable en escenarios reales.

### Gobernanza del release

Quedan oficialmente establecidos:

- `docs/SEMVER_POLICY.md`
- `docs/RELEASE_WORKFLOW.md`
- `docs/RELEASE_CANDIDATE_CHECKLIST.md`
- `docs/MIGRATION_COMPATIBILITY.md`
- `CHANGELOG.md`

Estos artefactos convierten la publicación de la librería en un proceso gobernado, trazable y repetible.

## Migración

No se requieren cambios de código para consumidores que vienen de `0.0.23`.

Se recomienda validar visualmente y funcionalmente los flujos que usan:

- `DataView`
- `OrderList`
- `PickList`
- `Tree`

Ver también:

- `docs/MIGRATION_COMPATIBILITY.md`
