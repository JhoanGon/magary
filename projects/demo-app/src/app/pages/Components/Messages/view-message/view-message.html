MagaryMessage

Componente para mostrar mensajes y alertas en línea.

Import

Importa el componente en tu módulo o componente standalone.

Severidades

Diferentes niveles de severidad para comunicar el estado.

Cerrable

Mensajes que pueden ser descartados por el usuario.

Icono Personalizado

Usa cualquier clase de icono compatible (ej: FontAwesome).

Animaciones

Entrada y Salida suave utilizando Angular Animations.

@for (msg of msgs; track $index) { }

Propiedades (Inputs)

Todas las propiedades disponibles del componente.

Nombre Tipo Valor por Defecto Descripción
severity 'success' | 'info' | 'warn' | 'error' | 'secondary' | 'contrast' 'info' Nivel de severidad del mensaje.
text string undefined Texto principal (resumen) del mensaje.
detail string undefined Texto detallado del mensaje.
icon string undefined Clase CSS del icono personalizado.
closable boolean false Si es verdadero, muestra un botón de cerrar.
life number undefined Tiempo en milisegundos para cerrar automáticamente el mensaje.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onClose EventEmitter<Event> Se emite cuando el usuario cierra el mensaje.
