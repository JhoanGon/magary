Características Avanzadas

Ejemplos de arrastre, redimensionamiento y posicionamiento.

Draggable

Este diálogo se puede arrastrar desde el encabezado. Nota que [modal]="false" permite interactuar con el fondo.

Resizable

Este diálogo tiene un manejador en la esquina inferior derecha.

Custom Position

Este diálogo usa position="right" y glass="true". Ya no necesitas estilos CSS manuales complejos.

Glass Effect

Este diálogo utiliza el efecto Glass nativo. Combina desenfoque de fondo con transparencia ajustada.

Maximizable (Container)

El diálogo se renderiza dentro de este contenedor y se maximiza solo aquí (usando appendTo="local").

Container Dialog

Soy libre, pero solo dentro de este contenedor.

Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
header string undefined Título del diálogo.
visible boolean false Controla la visibilidad. Importante: Usar [(visible)]="var" para que la variable se actualice a false al cerrar. Si usas solo [visible], la variable quedará en true desincronizada.
modal boolean true Si es true, muestra una máscara que bloquea el fondo.
maximizable boolean false Habilita el botón y la funcionalidad de maximizar.
dismissableMask boolean false Si es true, cierra el diálogo al hacer click fuera.
draggable boolean true Habilita el arrastre del diálogo desde el encabezado.
resizable boolean false Habilita el redimensionamiento del diálogo.
appendTo 'body' | 'local' 'body' Define dónde se adjunta el diálogo. 'local' permite restringirlo a un contenedor relativo.
position string 'center' Posición del diálogo: 'center', 'top', 'bottom', 'left', 'right', 'top-left', 'top-right', 'bottom-left', 'bottom-right'.
width string undefined Ancho del diálogo (ej. '50vw', '300px').
height string undefined Alto del diálogo (ej. '50vh', 'auto').
backgroundColor string undefined Color de fondo personalizado del diálogo.
glass boolean false Aplica un efecto de Glassmorphism (desenfoque y transparencia) al diálogo.
