MagaryTooltip

Directiva para mostrar información contextual flotante.

Importar

Módulos necesarios para utilizar este componente.

Nota: Para que el Tooltip se visualice correctamente, debes agregar los estilos globales en tu angular.json bajo la propiedad styles:

O importarlo en tu styles.scss global.

Posicionamiento

Soporta las 4 direcciones principales: top, bottom, left, right.

Propiedades (Inputs)

Configuración de la directiva.

Nombre Tipo Defecto Descripción
magaryTooltip string undefined Texto a mostrar en el tooltip.
tooltipPosition "top" | "bottom" | "left" | "right" "top" Posición relativa al elemento.
