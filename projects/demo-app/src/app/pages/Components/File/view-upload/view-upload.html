Propiedades (Inputs)

Todas las propiedades disponibles del componente.

Nombre Tipo Valor por Defecto Descripción
mode 'basic' | 'advanced' 'basic' Modo de visualización del componente.
multiple boolean false Permite seleccionar múltiples archivos.
accept string '*' Tipos de archivo permitidos (ej: 'image/*').
maxFileSize number null Tamaño máximo de archivo en bytes.
url string '' URL del endpoint de carga (opcional).
name string 'file' Nombre del campo en el FormData.
withCredentials boolean false Enviar cookies y headers de autenticación.
chooseLabel string 'Choose' Texto del botón de selección.
uploadLabel string 'Upload' Texto del botón de carga (modo avanzado).
cancelLabel string 'Cancel' Texto del botón de cancelar (modo avanzado).

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onSelect EventEmitter<UploadEvent> Se emite cuando se seleccionan archivos.
onUpload EventEmitter<UploadEvent> Se emite cuando la carga se completa exitosamente.
onError EventEmitter<{{ "{" }} error: string {{ "}" }}> Se emite cuando ocurre un error (ej: tamaño excedido).
onClear EventEmitter<void> Se emite cuando se limpia la lista de archivos.
