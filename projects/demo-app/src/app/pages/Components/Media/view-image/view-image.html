MagaryImage

Componente para mostrar imágenes con funcionalidades avanzadas como previsualización, zoom y rotación.

Import

Importa el componente en tu módulo o componente standalone.

Propiedades (Inputs)

Todas las propiedades disponibles del componente.

Nombre Tipo Valor por Defecto Descripción
src string undefined URL de la imagen.
alt string undefined Texto alternativo para la imagen.
width string undefined Ancho de la imagen (ej: '100%', '250px').
height string undefined Alto de la imagen.
preview boolean false Habilita el modo de previsualización al hacer click.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onLoad EventEmitter<Event> Se emite cuando la imagen se carga exitosamente.
onError EventEmitter<Event> Se emite si ocurre un error al cargar la imagen.

Accesibilidad

Consideraciones para lectores de pantalla y navegación por teclado.

  • Texto Alternativo: Utiliza siempre la propiedad alt para describir la imagen.
  • Teclado: En modo previsualización, los controles de zoom y rotación son accesibles por teclado.
