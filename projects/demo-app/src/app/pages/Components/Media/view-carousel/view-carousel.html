Propiedades (Inputs)

Principales propiedades disponibles en magary-carousel.

Nombre Tipo Por Defecto Descripcion
value unknown[] [] Items que renderiza el carrusel.
numVisible number 1 Cantidad de items visibles.
numScroll number 1 Cantidad de items por desplazamiento.
circular boolean false Activa navegacion circular.
responsiveOptions CarouselResponsiveOptions[] [] Configuracion responsive por breakpoints.
indicatorStyle CarouselIndicatorStyle 'dots' Estilo de indicadores (dots, lines, fraction, etc.).
navPosition CarouselNavPosition 'default' Posicion de los navegadores.
autoplayInterval number 0 Intervalo en ms para autoplay (0 desactiva).
showNavigators boolean true Muestra botones de anterior/siguiente.
showIndicators boolean true Muestra indicadores de pagina.
effect CarouselEffect 'slide' Efecto de transicion.
transitionDuration number 300 Duracion de la animacion en ms.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripcion
pageChange number Se emite cuando cambia la pagina activa.
slideChange CarouselSlideEvent<T> Se emite antes del cambio de slide.
slideChanged CarouselSlideEvent<T> Se emite despues del cambio de slide.
itemClick { item: T; index: number } Se emite al hacer click sobre un item.
autoplayStart void Se emite cuando inicia autoplay.
autoplayStop void Se emite cuando se detiene autoplay.
