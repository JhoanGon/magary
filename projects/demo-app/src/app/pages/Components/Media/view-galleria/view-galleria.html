Galleria

Componente avanzado para mostrar galerías de imágenes con thumbnails y navegación.

Importación

Básico (Thumbnails Abajo)

Configuración estándar con tira de imágenes en la parte inferior.

Thumbnails a la Izquierda

La tira de imágenes se posiciona verticalmente a la izquierda.

Indicadores (Sin Thumbnails)

Navegación mediante indicadores circulares superpuestos.

AutoPlay & Circular

Pase de diapositivas automático con bucle infinito.

Propiedades (Inputs)

Nombre Tipo Predeterminado Descripción
value unknown[] [] Array de objetos a mostrar.
showThumbnails boolean true Muestra la cinta de miniaturas.
showIndicators boolean false Muestra los indicadores de página.
autoPlay boolean false Inicia el pase de diapositivas automáticamente.
itemTemplateRef TemplateRef null Referencia a la plantilla del item principal.
thumbnailTemplateRef TemplateRef null Referencia a la plantilla de la miniatura.
width string 100% Ancho máximo del componente (ej. '640px').
