Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
label string undefined Texto del botón.
icon string undefined Clase del icono (ej: 'fas fa-home').
iconPos 'left' | 'right' 'left' Posición del icono.
severity 'primary' | 'secondary' | 'success' | 'info' | 'warning' | 'danger' | 'help' 'primary' Color semántico del botón.
variant 'solid' | 'outlined' | 'text' 'solid' Estilo visual del botón.
size 'small' | 'normal' | 'large' 'normal' Tamaño del botón.
disabled boolean false Deshabilita el botón.
loading boolean false Muestra un indicador de carga.
rounded boolean false Bordes completamente redondeados.
shadow number 0 Nivel de sombra (1-5).

Accesibilidad

Consideraciones para lectores de pantalla y navegación por teclado.

  • Teclado: El botón es completamente navegable mediante la tecla Tab y activable con Enter o Espacio.
  • ARIA: Se utilizan atributos ARIA adecuados, incluyendo aria-label si no hay texto visible.
  • Foco: El estado de foco es visible y claro para la navegación por teclado.
  • Contraste: Los colores por defecto cumplen con los estándares de contraste WCAG AA.
