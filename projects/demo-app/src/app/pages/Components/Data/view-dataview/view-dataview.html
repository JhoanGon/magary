DataView

DataView muestra datos en formato de cuadrícula o lista con paginación y personalización.

Importación

Ejemplo Básico

Ejemplo de DataView con layout conmutable entre lista y cuadrícula.
{{ product.name }}
{{ product.category }}
\${{ product.price }}
{{ product.category }}
{{ product.name }}
\${{ product.price }}

Propiedades (Inputs)

Nombre Tipo Predeterminado Descripción
value any[] [] Array de objetos a mostrar.
layout 'list' | 'grid' 'list' Diseño de visualización de los datos.
paginator boolean false Habilita el paginador.
rows number 0 Número de filas par página.
emptyMessage string 'No records found' Mensaje a mostrar cuando no hay datos.

Eventos (Outputs)

Nombre Tipo Descripción
onPage EventEmitter<PaginatorState> Callback invocado cuando cambia la página.
