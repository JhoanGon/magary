Propiedades (Inputs)

Atributos para controlar el comportamiento del componente.

Nombre Tipo Por Defecto Descripción
value TreeNode[] [] Array de nodos del árbol a mostrar.
selectionMode 'single' | 'multiple' | 'checkbox' null Define el modo de selección.
selection MagaryTreeNode | MagaryTreeNode[] | Record<string, boolean> | null null Enlace para el/los nodo(s) seleccionado(s).
filterPlaceholder string 'Search...' Texto del input de filtrado.
filterMode 'lenient' | 'strict' 'lenient' Define si el filtro es parcial o exacto.
filter boolean false Habilita la barra de búsqueda.
draggable boolean false Permite arrastrar nodos.
droppable boolean false Permite soltar nodos.
validateDrop boolean false Valida drop para evitar soltar sobre si mismo/descendientes.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onNodeSelect EventEmitter<MagaryTreeNodeSelectionEvent> Callback al seleccionar un nodo.
onNodeUnselect EventEmitter<MagaryTreeNodeSelectionEvent> Callback al deseleccionar un nodo.
onNodeExpand EventEmitter<MagaryTreeNode> Callback al expandir un nodo.
onNodeCollapse EventEmitter<MagaryTreeNode> Callback al colapsar un nodo.
onNodeDrop EventEmitter<MagaryTreeNodeDropEvent> Callback de drag and drop con parent y dragNode.
