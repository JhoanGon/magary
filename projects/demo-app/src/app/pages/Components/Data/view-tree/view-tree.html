Magary Tree

Magary Tree muestra datos jerárquicos con soporte para selección.

Ejemplo Básico

Árbol estándar con selección única.

Seleccionado: {{ selectedFile.label }}

Filtrado

Búsqueda integrada para filtrar nodos.

Drag & Drop

Reordenamiento de nodos mediante arrastrar y soltar.

Propiedades (Inputs)

Atributos para controlar el comportamiento del componente.

Nombre Tipo Por Defecto Descripción
value TreeNode[] [] Array de nodos del árbol a mostrar.
selectionMode 'single' | 'multiple' | 'checkbox' null Define el modo de selección.
selection any null Enlace para el/los nodo(s) seleccionado(s).
loading boolean false Muestra un estado de carga (skeleton).
filter boolean false Habilita la barra de búsqueda.
draggable boolean false Permite arrastrar nodos.
droppable boolean false Permite soltar nodos.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onNodeSelect EventEmitter<any> Callback al seleccionar un nodo.
onNodeUnselect EventEmitter<any> Callback al deseleccionar un nodo.
onNodeExpand EventEmitter<any> Callback al expandir un nodo.
onNodeCollapse EventEmitter<any> Callback al colapsar un nodo.
