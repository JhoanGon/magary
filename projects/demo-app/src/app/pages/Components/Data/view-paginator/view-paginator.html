Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
totalRecords number 0 Total de registros a paginar.
rows number 10 Número de filas a mostrar por página.
first number 0 Índice del primer registro a mostrar.
rowsPerPageOptions number[] [] Array de opciones para el selector de filas por página.
pageLinkSize number 5 Número de enlaces a páginas a mostrar.
showPageLinks boolean true Si se deben mostrar los enlaces a páginas.

Eventos (Outputs)

Eventos emitidos por el componente

Nombre Tipo Descripción
onPageChange EventEmitter<PaginatorState> Callback que se invoca cuando cambia la página.
