Propiedades (Inputs)

Atributos para controlar el comportamiento del componente.

Nombre Tipo Por Defecto Descripción
value any[] [] Array de eventos a mostrar.
layout 'vertical' | 'horizontal' 'vertical' Orientación del timeline.
align 'left' | 'right' | 'top' | 'bottom' | 'alternate' 'left' Alineación del contenido y posición.

Templates

Templates disponibles para personalizar el contenido.

Nombre Contexto Descripción
content $implicit: item Template para el contenido principal del evento.
opposite $implicit: item Template para el lado opuesto (ej. fecha).
marker $implicit: item Template para el marcador central.
