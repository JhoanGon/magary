Nombre Tipo Default Descripción

API Reference

Inputs para magary-radio.

Nombre Tipo Default Descripción
value unknown - Valor del radio button.
name string - Nombre del grupo de radios (atributo HTML).
label string - Texto descriptivo opcional.
inputId string - ID del input interno (para labels).
disabled boolean false Deshabilita el radio button (soporta atributo HTML).
