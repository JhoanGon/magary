CascadeSelect

CascadeSelect muestra una estructura anidada de opciones.

Importar

Importa el componente en el componente standalone.

Uso Basico

Comportamiento estandar donde solo los nodos hoja (ciudades) son seleccionables.

Solo Selección de Hojas

Selección de Grupo

Permite seleccionar grupos padres (Países o Estados) como valores válidos.

Propiedades (Inputs)

Propiedades disponibles para el componente.

Nombre Tipo Por Defecto Descripción
options unknown[] null Un array de objetos para mostrar como las opciones disponibles.
optionLabel string label Nombre del campo de etiqueta de una opción.
optionValue string null Nombre del campo de valor de una opción. Por defecto es el objeto de la opción en sí.
optionGroupLabel string null Nombre del campo de etiqueta de un grupo de opciones.
optionGroupChildren string[] ['children'] Nombres de las propiedades a usar como opciones hijas de un grupo de opciones.
optionGroupSelectable boolean false Cuando está habilitado, permite la selección de grupos de opciones (padres).
placeholder string Select an option Texto por defecto para mostrar cuando no hay ninguna opción seleccionada.
disabled boolean false Cuando está presente, especifica que el componente debe estar deshabilitado.
loading boolean false Muestra spinner en el trigger y bloquea interacción.
size 'small' | 'normal' | 'large' 'normal' Tamaño visual del control.
invalid boolean false Activa estado visual de error.
error string '' Mensaje de error mostrado debajo del control.
helpText string '' Texto de ayuda cuando no hay error.
width string 100% Ancho del componente.
