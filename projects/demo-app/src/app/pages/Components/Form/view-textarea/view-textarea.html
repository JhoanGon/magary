Magary TextArea

Area de texto con autoredimensionamiento y soporte CVA.

Importación

Importa el módulo en tu componente standalone.

Uso Básico

Textarea simple y con AutoResize.

Character Counter

Contador de carácteres y límite de input.

API Reference

Inputs para magary-textarea.

Nombre Tipo Default Descripción
rows number 3 Filas iniciales.
cols number 20 Columnas iniciales.
placeholder string - Texto de ayuda.
autoResize boolean false Si es true, la altura se ajusta automáticamente al contenido.
maxlength number - Límite de caracteres.
showCounter boolean false Muestra el contador `current / max`.
disabled boolean false Deshabilita el textarea.
