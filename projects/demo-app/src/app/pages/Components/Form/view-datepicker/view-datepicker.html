Magary DatePicker

Selector de fecha intuitivo con calendario popup.

Importación

Importa el módulo en tu componente standalone.

Asegúrate de importar también FormsModule para usar [(ngModel)].

Uso Básico

Selección de fecha simple.

Selected: {{ date | date: "fullDate" }}

Advanced Features

Navegación inteligente (Año/Mes), Input manual y Rangos.

Range Selection

Selecciona un rango de fechas (Inicio y Fin).

API Reference

Inputs para magary-datepicker.

Nombre Tipo Default Descripción
ngModel Date | Date[] null Vincula el valor de la fecha o rango.
selectionMode 'single' | 'range' single Modo de selección.
minDate Date - Fecha mínima permitida.
maxDate Date - Fecha máxima permitida.
placeholder string - Texto de ayuda.
disabled boolean false Deshabilita el selector.
