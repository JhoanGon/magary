API

Propiedades y eventos disponibles

Nombre Tipo Default Descripción
checked boolean false Valor del checkbox (Two-way binding)
label string '' Texto a mostrar junto al checkbox
disabled boolean false Deshabilita la interacción
color 'primary' | 'success' | ... 'primary' Color semántico del componente
required boolean false Marca el campo como requerido
change EventEmitter<boolean> - Evento emitido al cambiar el valor

Accesibilidad

  • Keyboard Support Soporta navegación con Tab y activación con Space.
  • Screen Readers Utiliza un input nativo oculto para garantizar compatibilidad con lectores de pantalla.
  • Labels El texto del label está asociado programáticamente con el input mediante el atributo for.
