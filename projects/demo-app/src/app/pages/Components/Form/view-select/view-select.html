Magary Select

Componente de selección nativo estilizado compatible con Reactive Forms y soporte para objetos complejos.

Importación

Importa el módulo en tu componente standalone.

Uso Básico

Select simple con opciones primitivas o de objetos.

Reactive Forms & Objects

Integración completa con `FormControl` y manejo de objetos complejos.

Filtering & Clear

Habilita búsqueda por texto en listas largas y botón de limpieza. El filtrado se realiza localmente sobre la propiedad `optionLabel`.

API Reference

Inputs y Outputs disponibles para magary-select.

Nombre Tipo Default Descripción
options unknown[] [] Array de objetos o primitivos a mostrar.
optionLabel string - Nombre de la propiedad a usar como etiqueta.
optionValue string - Nombre de la propiedad a usar como valor.
placeholder string Select... Texto a mostrar cuando no hay selección.
disabled boolean false Deshabilita el componente (soporta atributo HTML).
loading boolean false Muestra un spinner de carga (soporta atributo HTML).
filter boolean false Habilita el input de búsqueda.
showClear boolean false Muestra botón para limpiar selección.
