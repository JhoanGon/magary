Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
label string undefined Etiqueta del input.
placeholder string undefined Texto de ayuda temporal.
value string '' Valor del input.
type string 'text' Tipo de input (text, email, password, etc).
size 'small' | 'normal' | 'large' 'normal' Tamaño del componente.
variant 'outlined' | 'filled' | 'underlined' 'outlined' Estilo visual del input.
disabled boolean false Deshabilita el input.
readonly boolean false Modo solo lectura.
required boolean false Marca el campo como obligatorio.
error string undefined Mensaje de error.
success boolean false Indica estado de éxito.
loading boolean false Muestra indicador de carga.
prefixIcon string undefined Clase del icono inicial (ej: 'user').
suffixIcon string undefined Clase del icono final.

Eventos (Outputs)

Eventos emitidos por el componente

Nombre Tipo Descripción
valueChange EventEmitter<string> Se emite cuando cambia el valor.
iconClick EventEmitter<'prefix' | 'suffix'> Se emite al hacer click en un icono.
inputFocus EventEmitter<FocusEvent> Se emite al recibir foco.
inputBlur EventEmitter<FocusEvent> Se emite al perder foco.

Accesibilidad

Consideraciones para lectores de pantalla y navegación por teclado.

  • Teclado: El input es completamente navegable mediante la tecla Tab.
  • Labels: Se asocia correctamente el label con el input mediante el atributo for.
  • ARIA: Se utilizan atributos ARIA para indicar estados como inválido o requerido.
  • Foco: El estado de foco es visible y claro.
