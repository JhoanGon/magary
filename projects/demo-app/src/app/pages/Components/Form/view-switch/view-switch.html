MagarySwitch

Componente de interruptor para alternar entre dos estados, con animaciones fluidas y soporte completo de temas.

Import

Importa el componente en tu módulo o componente standalone

Ejemplo Básico

Default

Value: {{ checked1() }}

Estados

Diferentes estados interactivos del componente

Colores

Variantes semánticas de color

API

Propiedades y eventos disponibles

Nombre Tipo Default Descripción
checked boolean false Valor del switch (Two-way binding)
label string '' Texto a mostrar junto al switch
disabled boolean false Deshabilita la interacción
color 'primary' | 'success' | ... 'primary' Color semántico del componente
change EventEmitter<boolean> - Evento emitido al cambiar el valor

Accesibilidad

  • Role Utiliza role="switch" para identificar el componente ante tecnologías de asistencia.
  • Keyboard Support Soporta navegación con Tab y activación con Space.
  • ARIA Attributes Gestiona automáticamente aria-checked para reflejar el estado actual.
