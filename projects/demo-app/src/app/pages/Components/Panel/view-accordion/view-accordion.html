MagaryAccordion

Agrupa una colección de contenidos en paneles colapsables.

Básico

Accordion permite expandir un solo panel a la vez por defecto.

Propiedades (Inputs)

Configuración del componente Accordion y AccordionTab.

MagaryAccordion

Nombre Tipo Defecto Descripción
multiple boolean false Permite abrir múltiples paneles.

MagaryAccordionTab

Nombre Tipo Defecto Descripción
header string undefined Título del panel.
disabled boolean false Deshabilita el panel.
selected boolean false Estado de apertura del panel.

Eventos (Outputs)

Eventos emitidos por MagaryAccordion.

Nombre Tipo Descripción
onOpen EventEmitter Se emite al abrir un tab.
onClose EventEmitter Se emite al cerrar un tab.
