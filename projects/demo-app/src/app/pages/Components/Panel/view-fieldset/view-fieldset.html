Fieldset

Fieldset es un componente de agrupación con un área de contenido y una leyenda.

Importación

Básico

Propiedades (Inputs)

Nombre Tipo Predeterminado Descripción
legend string null Texto del encabezado.
toggleable boolean false Cuando está habilitado, el contenido se puede alternar.
collapsed boolean false Define el estado inicial colapsado.

Eventos (Outputs)

Nombre Tipo Descripción
onBeforeToggle EventEmitter<unknown> Callback que se invoca antes de cambiar el estado de colapso.
onAfterToggle EventEmitter<unknown> Callback que se invoca después de cambiar el estado de colapso.
