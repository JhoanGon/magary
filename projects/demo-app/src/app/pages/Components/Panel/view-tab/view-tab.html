Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
lineColor string '#06b6d4' Color de la línea que indica la pestaña activa
activeBg string | null null Color de fondo para la pestaña activa.
activeText string | null null Color de texto para la pestaña activa.
hoverBg string | null null Color de fondo al pasar el cursor sobre cada pestaña.
positionContent string 'center' Posición del contenido, se utiliza los mismos que con justify-content
panelWidth 'auto' | 'full' 'full' Controla el ancho del panel activo. El valor por defecto 'full' evita huecos visuales y estira el contenido al ancho disponible.
tabListAriaLabel string 'Tabs' Etiqueta accesible para el contenedor tablist.
backgroundLine / activeTabBackground / activeTabTextColor string | null null Inputs legacy soportados por compatibilidad (preferir lineColor, activeBg y activeText).

Eventos (Outputs)

Eventos emitidos por el componente

Este componente no emite eventos directamente, pero puedes reaccionar a los cambios en las pestañas internas.

Contenido Esperado

Este componente espera recibir uno o más componentes <magary-tab> como hijos.

  • Cada <magary-tab> debe tener una propiedad label.
  • El contenido dentro de cada tab se mostrará solo cuando esté activa.

Accesibilidad

Características de accesibilidad implementadas

  • Teclado: Soporta Tab, ArrowLeft, ArrowRight, Home y End
  • Activación: Las pestañas se activan con Enter o Click
  • Foco: Las pestañas muestran foco visual
  • Semántica: Botones con estructura clara y etiquetas visibles
