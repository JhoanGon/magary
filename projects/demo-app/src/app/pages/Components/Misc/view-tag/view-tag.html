MagaryTag

Etiquetas para categorizar contenido o indicar estado.

Estados & Severidad

Varios estados predefinidos para comunicar semántica (éxito, error, info, etc.).

Redondeado & Iconos

Opcionalmente bordes completamente redondeados (píldoras) e iconos integrados.

Propiedades (Inputs)

Configuración del componente Tag.

Nombre Tipo Defecto Descripción
value string undefined Texto a mostrar.
severity "success" | "info" | "warning" | "danger" | "secondary" | "contrast" "info" Color semántico de la etiqueta.
rounded boolean false Si es verdadero, redondea completamente los bordes.
icon string undefined Clase del icono a mostrar.
