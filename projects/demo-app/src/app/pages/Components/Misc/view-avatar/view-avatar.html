Propiedades (Inputs)

Todas las propiedades disponibles del componente:

Nombre Tipo Valor por Defecto Descripción
label string '' Texto para generar iniciales automáticamente
image string '' URL de la imagen del avatar
icon string '' Ícono FontAwesome (sin prefijo 'fa-')
shape 'circle' | 'square' 'square' Forma del avatar
size 'normal' | 'large' | 'xlarge' 'normal' Tamaño del avatar
badgeValue string '' Valor del badge de notificación
badgeSeverity 'info' | 'success' | 'warning' | 'danger' 'danger' Color/tipo del badge
customStyle Record<string, any> - Estilos CSS personalizados

Eventos (Outputs)

Eventos emitidos por el componente:

Nombre Tipo Descripción
avatarClick AvatarClickEvent Se emite cuando se hace click en el avatar o badge

Accesibilidad

  • ARIA Labels: Etiquetas descriptivas automáticas
  • Navegación por teclado: Soporte para Enter y Espacio
  • Focus visible: Indicadores visuales claros
  • Alto contraste: Colores accesibles por defecto
  • Texto alternativo: Fallbacks apropiados para imágenes
