Import the component to use it in your application.

Basic usage with different dimensions.

Shapes

Card Example

A complex example simulating a card loading state.

API

Properties and events available for the component.

Name Type Default Description
shape string rectangle Shape of the element. Options: 'rectangle' | 'circle'.
size string null Size of the element, shortcut for width and height.
width string 100% Width of the element.
height string 1rem Height of the element.
borderRadius string null Border radius of the element.
animation string shimmer Type of the animation. Options: 'shimmer' | 'none'.
