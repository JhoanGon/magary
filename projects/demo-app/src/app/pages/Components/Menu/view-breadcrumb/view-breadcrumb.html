MagaryBreadcrumb

Indica la ubicación de la página actual dentro de una jerarquía de navegación.

Ejemplo Básico

Uso estándar con un icono de inicio y rutas.

Propiedades (Inputs)

Atributos para configurar el componente.

Nombre Tipo Por Defecto Descripción
model BreadcrumbItem[] null Array de items del menú.
home BreadcrumbItem null Item inicial, usualmente el home.
style object null Estilos inline del componente.
styleClass string null Clase CSS externa.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripción
onItemClick EventEmitter Callback invocado al hacer click en un item.
