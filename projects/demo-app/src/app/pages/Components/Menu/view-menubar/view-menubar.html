Magary Menubar

Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Por Defecto Descripción
model MenuItem[] [] Array de objetos MenuItem para renderizar el menú.
style object null Estilos en línea.
styleClass string '' Clase CSS personalizada.

Eventos (Outputs)

Eventos gestionados a través del modelo MenuItem

Nombre Tipo Descripción
command (event: { originalEvent: Event, item: MenuItem }) => void Callback que se ejecuta cuando se hace click en un item del menú. Defínase dentro del objeto MenuItem.
