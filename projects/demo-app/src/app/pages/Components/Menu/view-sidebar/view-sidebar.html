Vista Catalogo

Esta variante usa menu + menuTitle en lugar de sections. En este demo lo dejamos sin colapso para que el patron de navegacion se entienda mejor.

  • Modo compacto para apps administrativas.
  • Logo de marca configurable por posicion.
  • Bloque de usuario reutilizable.

Propiedades (Inputs)

Inputs principales para configurar comportamiento y look & feel.

Nombre Tipo Default Descripcion
sections SidebarSection[] [] Modo por secciones (usa paneles independientes).
menu MenuItem[] [] Modo menu unico cuando sections esta vacio.
menuTitle string 'Menu' Titulo para el modo menu.
showUserSection boolean false Muestra bloque inferior con avatar y logout.
userName string 'John Doe' Nombre mostrado en el bloque de usuario.
userEmail string 'user@example.com' Email mostrado en el bloque de usuario.
avatarConfig AvatarConfig {{ '{' }} type: 'label' {{ '}' }} Configura avatar (tipo, shape, size, badge, etc).
showLogo boolean true Muestra logo principal en header.
logoSrc string 'assets/logo.svg' Ruta del logo principal.
appTitle string 'PRIMENG' Titulo de la aplicacion en header.
showBrandLogo boolean false Muestra logo de marca en top/bottom/center.
brandLogoPosition 'top' | 'bottom' | 'center' 'bottom' Posicion del logo de marca.
menuBackgroundColor string 'var(--surface-50)' Fondo de los paneles de menu.
menuTextColor string 'var(--text-primary)' Color de texto de items y titulos.
menuHoverColor string 'var(--surface-100)' Color hover para items interactivos.
collapsible boolean true Permite colapsar/expandir sidebar en desktop.
showToggle boolean true Muestra boton de toggle en header.

Eventos (Outputs)

Eventos emitidos por el componente.

Nombre Tipo Descripcion
onLogout void Se emite cuando el usuario pulsa "Cerrar sesion".
closeSidebar void Se emite cuando el sidebar mobile se cierra (backdrop, boton close o cierre programatico).

Estructura Esperada

Tipos utiles para poblar el sidebar correctamente.

  • SidebarSection: title, icon, items y overrides opcionales de color.
  • MenuItem: label, icon, route, children, disabled.
  • AvatarConfig: type (image, label, icon) mas size/shape/badge.

Accesibilidad y Responsive

Recomendaciones practicas para una UX solida.

  • Mobile: usa el boton hamburguesa y backdrop para cierre rapido.
  • Navegacion: define labels cortos y jerarquia clara en cada seccion.
  • Contraste: valida colores de menuTextColor/menuHoverColor.
  • Densidad: en sidebars largos combina secciones y submenu para reducir scroll.
