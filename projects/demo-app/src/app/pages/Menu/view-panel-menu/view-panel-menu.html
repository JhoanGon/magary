Propiedades (Inputs)

Estas son las entradas que puedes configurar en el componente:

Nombre Tipo Valor por Defecto Descripción
title string 'Panel Menu' Título del panel que se muestra en el header
items MenuItem[] [] Lista de elementos que componen el menú
backgroundColor string '#f9fafb' Color de fondo del panel
textColor string '#1f2937' Color del texto
hoverColor string '#007bff' Color de fondo al pasar el mouse sobre un ítem
borderRadius string '8px' Radio del borde del panel
shadow number 0 Nivel de sombra del panel
width string '100%' Ancho del panel

Eventos

Este componente no emite eventos personalizados directamente.

Estructura esperada

  • El items debe ser un arreglo de objetos MenuItem.
  • Cada MenuItem puede tener children de forma recursiva.
  • Cada ítem puede tener un label, route y un icon.

Accesibilidad

  • Teclado: La navegación del menú puede extenderse con soporte de accesibilidad de teclado.
  • Visibilidad: Cambios de color y foco en hover para mejor UX.
  • Semántica: Uso de elementos como <a> para enlaces semánticos.
