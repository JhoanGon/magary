Propiedades (Inputs)

Estas son las entradas que puedes configurar en el componente:

Nombre Tipo Valor por Defecto Descripción
label string undefined Texto que se muestra si no hay imagen ni ícono
image string undefined URL de la imagen del avatar
icon string undefined Ícono FontAwesome que se mostrará si no hay imagen
shape 'circle' | 'square' 'square' Forma del avatar
size 'normal' | 'large' | 'xlarge' 'normal' Tamaño del avatar
badgeValue string undefined Valor del badge (por ejemplo, cantidad de notificaciones)
badgeSeverity 'info' | 'success' | 'warning' | 'danger' 'danger' Color del badge
customStyle {{ "{" }} [key: string]: any {{ "}" }} {{ "{" }}{{ "}" }} Estilos personalizados en línea

Eventos

Este componente no emite eventos personalizados.

Accesibilidad

  • Contraste: El componente utiliza contraste alto por defecto para íconos y texto.
  • Fallbacks: Muestra texto si no se proporciona imagen ni ícono, garantizando visibilidad.
  • Semántica: Uso apropiado de etiquetas <img> e íconos como contenido visual.
