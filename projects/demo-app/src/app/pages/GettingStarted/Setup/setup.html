Configuración

Guía paso a paso para configurar ng-magary en tu proyecto.

1

Estilos y Tema

ng-magary utiliza variables CSS nativas para el theming y Glassmorphism. Copia el siguiente bloque de variables en tu archivo styles.scss global para habilitar el diseño premium.

2

Iconos

La librería utiliza FontAwesome para los iconos. Instálalo en tu proyecto usando npm y luego agrégalo a tu angular.json en la sección de styles.

3

Animaciones

Para que las animaciones (como ripples y transiciones) funcionen, asegúrate de proveer el módulo de animaciones en tu app.config.ts (Standalone) o app.module.ts.
