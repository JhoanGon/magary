Theming System

Un sistema de temas flexible y potente basado en CSS Variables y Signals. Personaliza cada aspecto de tu aplicación con facilidad.

Introducción

Magary utiliza un enfoque moderno para el theming, aprovechando las CSS Custom Properties (Variables) para definir colores, espaciados y tipografía. Esto permite cambiar de tema en tiempo real sin necesidad de recargar la página y con un rendimiento óptimo.

El sistema está gestionado por MagaryThemeService, que se encarga de persistir la preferencia del usuario y aplicar las clases necesarias.

Creando un Nuevo Tema

Para añadir un nuevo tema (por ejemplo, "Midnight Purple"), simplemente define un bloque CSS con el atributo data-theme correspondiente. Debes sobrescribir las variables de color principales.

Variables Principales

Estas son las variables clave que debes definir para asegurar que todos los componentes se adapten correctamente:

Primary --primary-500
Surface --surface-0
Text Main --text-primary
Text Muted --text-secondary
