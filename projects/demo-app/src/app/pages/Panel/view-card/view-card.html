Propiedades (Inputs)

Todas las propiedades disponibles del componente

Nombre Tipo Valor por Defecto Descripción
img string undefined URL de la imagen a mostrar
positionImage 'left' | 'right' | 'top' | 'bottom' 'top' Posición de la imagen respecto al contenido
shadow number 1 Nivel de sombra aplicado (0-5)
width string '250px' Ancho de la tarjeta
padding string '1rem' Espaciado interno de la tarjeta
gap string '1rem' Espacio entre elementos internos
borderRadius string '0.75rem' Radio del borde de la tarjeta
imageSize string '500px' Altura máxima de la imagen (para top/bottom)
backgroundColor string '#fff' Color de fondo de la tarjeta

Slots (Contenido)

Contenido personalizable dentro del componente

  • slot="header": Área del encabezado
  • slot="footer": Área del pie de página
  • Contenido libre entre ambos slots

Accesibilidad

Características de accesibilidad implementadas

  • Semántica: Estructura clara con imagen y contenido
  • Responsive: Imagen y contenido se adaptan a su posición
  • Contraste: Personalizable con backgroundColor
  • Imagen: Incluye atributo alt accesible
