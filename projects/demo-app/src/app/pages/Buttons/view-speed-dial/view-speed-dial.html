Propiedades (Inputs)

      
Todas las propiedades disponibles del componente

    

    

      

        
          

            Nombre
            Tipo
            Valor por Defecto
            Descripción
          

        
        
          

            items
            SpeedDialItem[]
            —
            Lista de acciones con ícono, tooltip y comando
          

          

            icon
            string
            'fas fa-plus'
            Ícono del botón inicial
          

          

            activeIcon
            string
            'fas fa-times'
            Ícono al estar abierto
          

          

            type
            
              'linear' | 'circle' | 'semicircle' | 'quartercircle'
            
            'linear'
            Tipo de disposición
          

          

            direction
            
              'up' | 'down' | 'left' | 'right' | 'up-left' | ...
            
            —
            Dirección del despliegue (solo para linear)
          

          

            radius
            number
            80
            Radio de expansión
          

          

            showMask
            boolean
            true
            Activa máscara al abrir
          

          

            background
            string
            '#007bff'
            Color del botón trigger
